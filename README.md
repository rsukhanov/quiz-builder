# App is running on the ```http://localhost:3000```


# How to Create a Sample Quiz after starting a servers

-Open http://localhost:3000 in your browser.  
-Click the "Create New Quiz" button.  
-Enter a Title 
-Click "Add Question"  
-Select type (Text Answer, True/False, or Multiple Choice).  
-Fill in the question text and correct answer.  
-Click "Save Quiz".
