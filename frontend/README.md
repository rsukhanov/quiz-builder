# Quiz Builder Frontend

Client-side application for creating and managing quizzes. Built with modern **Next.js 15 (App Router)**.

## Tech Stack

- **Framework:** Next.js 15
- **Language:** TypeScript 5
- **Styling:** Tailwind CSS 4
- **Forms & Validation:** React Hook Form + Zod
- **HTTP Client:** Axios
- **Icons:** Lucide React

## Getting Started

### 1. Prerequisites
Make sure the **Backend** is running on port `4200`.

### 2. Setup Environment
Run `npm install`

### 3. Start fronend server
Run `npm run dev`

```env
NEXT_PUBLIC_API_URL=http://localhost:4200
